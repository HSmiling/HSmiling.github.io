工具栏

File

选项 含义
Import Data 数据导入，其中包含数据集的导入以及平滑噪声数据
Create Session 创建会话，即创建当前要拟合的东西
Load Session 保存当前会话
Save Session 保存会话
Generate M-file 将当前结果生成.m文件
Print 输出
Print to Figure 输出为matlab的figure文件
Close Curve Fitting 关闭

View

选项 含义
Prediction Bounds 预测界限
Confidence Level 置信区间设置
Residuals 残差？
Clear Plot 清空图

Tools

选项 含义
New Custom Equation 使用自定义函数模拟图中散点数据的函数
Legend 右上角曲线标志
Grid 网格线
Zoom in 放大
Zoom Out 缩小
Pan 拖动页面移动
Axis Limit Control 调节坐标横纵轴显示
Default Axis Limits 恢复初始横纵坐标

Window

列表中显示的是当前打开的matlab窗口

Help

Curve Fitting Tool Help

Curve Fitting Toolbox Help：这里可以查看详细指南

About Curve Fitting

五大按钮

按钮 功能
DATA GUI - 导入、预览、命令、删除数据集
- 平滑噪声数据
FITTING GUI - 使用库或自定义方程、平滑样式曲线或插值拟合数据
- 检查并比较拟合结果，包括拟合系数值和拟合优度估计
- 持续跟踪当前对话中的所有拟合及相关数据集
EXCLUDE GUI - 标记从需要拟合中排除的数据
- 查看排除规则
PLOTTING GUI - 控制在cftool中数据集和拟合的显示
ANALYSIS GUI - 计算（插值或推断）、区分或合并一种拟合曲线
- 绘制分析结果和数据集

曲线拟合类型

Type of fit 解释
Custom Equations 用户自定义的函数类型
Exponential 指数逼近，有2种类型
Fourier 傅立叶逼近，有7种类型
Gaussian 高斯逼近，有8种类型
Interpolant 插值逼近，有4种类型：linear、nearest neighbor、cubic spline、shape-preserving
Polynomial 多项式逼近，有9种类型
Power 幂逼近，有2种类型
Rational 有理数逼近
Smoothing Spline 平滑样条
Sum of Sin Functions 正弦曲线逼近，有8种类型
Weibull 韦伯分布
